CSV rules

  • Point columns: lon/lat, longitude/latitude, or x/y.
  • To make polygons from point rows, include a shared group column such as polygon, group, Polygon_Name, or Name.
  • Rows are connected in file order within each group.
  • If a CSV has a WKT column with POLYGON(...), polygons are read directly.

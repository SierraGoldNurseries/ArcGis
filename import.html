1. Pick a project file

Supported backend files: .ppkx, .mpkx, .gdb.zip, .shp.zip, .geojson, .json, .kml, and CSV exports. ArcGIS packages are uploaded to GitHub and converted by the Actions workflow.

2. Backend connection

Best setup: deploy this site with the included backend function and set GitHub secrets as environment variables. Then users never paste a GitHub token into the browser.

Settings are saved only in this browser. The backend should use env vars: GITHUB_TOKEN, GITHUB_OWNER, GITHUB_REPO, GITHUB_BRANCH.

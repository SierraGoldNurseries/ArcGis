Enterprise fast mode: startup loads only simplified core block outlines. Irrigation, chemicals, and tree/detail files are deferred. Tree names like Chandler, RX1, VX211, and Clonal are never rendered as labels.
Web Worker enabled: large GeoJSON files are fetched, cached in IndexedDB, and parsed in a background thread so the map UI does not freeze.
Zoom rules:
Zoom < 15: outlines only
15+: block labels
17+: irrigation can display
19+: click a block for detail
